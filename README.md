# Documentation

Each machine contains:

- **README.md** — High-level overview of the assessment.
- **Executive Report** — Business-oriented summary of the engagement, findings, risks, and remediation recommendations.
- **Technical Report** — Detailed walkthrough of the assessment methodology, exploitation process, privilege escalation, and supporting evidence.
- **Supporting Artifacts** — Original outputs generated during the assessment, including Nmap scans, Gobuster results, Nikto reports, sqlmap logs, Burp Suite requests, screenshots, and other supporting files.
